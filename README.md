# Lab_2
 Blinky and Running LEDs 🟣 STM32 Programming with STM32F103C8T6 Blue Pill C++ in STM32 Cube IDE

For file setup follow lab 1 README.md Instructions

LAB2.hex is the compiled hexadecimal file generated from the STM32CubeIDE project. It contains the machine code that is loaded into the STM32 microcontroller in Proteus for simulation. By attaching the LAB2.hex file to the MCU in Proteus, the program executes exactly as it would on the actual hardware. This allows testing and verification of the circuit design before implementing it on the physical STM32 board.
